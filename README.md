
# Virtual Reality --> House

Blender + ThreeJS + WebGL

## Installation

  `$ npm install && cd public npm install && bower install && grunt build`

And then run:

  `$ npm start`

## Usage

Run --> http://localhost:5000


## Steps to navigate

```
c -> Aerial View
b -> Exit Aerial View

left/right/forward/backward
u -> Ascend
j -> Descend

On mouse click and move : rotate view
```
